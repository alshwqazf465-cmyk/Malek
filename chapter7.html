👻 Ghost Touch (اللمس الذاتي)

⚠️
Ghost Touch هو ظهور لمسات عشوائية على الشاشة بدون تدخل من المستخدم. يمكن أن يكون سببه سوفتوير أو هاردوير.

أسباب Ghost Touch:

السبب العرض الحل
شاحن رديء ظاهر فقط أثناء الشحن استخدم شاحن أصلي (5V/2A)
واقي شاشة رديء ظاهر دائماً أزل الواقي أو استخدم نوع جيد
كابل شاشة غير مثبت ظاهر بعد فتح الجهاز أعد تركيب كابل LCD بإحكام
Touch IC تالف ظاهر دائماً، يزداد مع الحرارة استبدال Touch IC (متقدم)
ماء/رطوبة ظاهر بعد تعرض للماء تنظيف بالكحول، تجفيف
Software Bug ظاهر بعد تحديث Factory Reset أو Rollback

تشخيص Ghost Touch:

1
افصل الشاحن — هل Ghost Touch يختفي؟
⬇
2
أزل واقي الشاشة — هل يختفي؟
⬇
3
ادخل Safe Mode — هل يختفي؟ (إذا نعم = تطبيق خاطئ)
⬇
4
جرب Factory Reset — هل يختفي؟ (إذا نعم = Software)
⬇
5
إذا بقي → مشكلة هاردوير (Touch IC أو كابل)

المشكلة السبب المحتمل الحل
شاشة سوداء (لكن الجهاز يعمل) كابل LCD، IC Display، Backlight أعد تركيب الكابل، فحص IC
خطوط عمودية/أفقية كابل LCD مكسور، IC Display استبدل كابل أو الشاشة
بقع صفراء/زرقاء ضغط على LCD، ماء، OLED حرق استبدل الشاشة
لمس لا يستجيب في منطقة Touch IC، كابل Touch أعد تركيب الكابل، استبدل IC
سطوع منخفض جداً Backlight IC، Ambient Sensor فحص IC إضاءة، استبدل Sensor
Flickering (وميض) كابل LCD، Software (PWM dimming) أعد تركيب الكابل، عطل DC Dimming
Dead Pixels عيب مصنعي أو ضرر فيزيائي استبدل الشاشة (لا يُصلح)

🔥 سكريبت اختبار الشاشة

screen-test.sh — اختبار شامل للشاشة 
#!/bin/bash
# Screen Test Script - Dead pixels, touch, colors

echo "[*] Screen Test Starting..."

# 1. Display Info
echo "[1] Display Info:"
adb shell dumpsys display | grep -E "DisplayDeviceInfo|width|height|density" | head -5

# 2. Touch Test (open touch test app)
echo "[2] Opening Touch Test App..."
adb shell am start -a android.intent.action.VIEW -d "https://touchscreen-test.com"

# 3. Color Test (flash colors)
echo "[3] Color Test - Flashing colors..."
for color in 0xFFFF0000 0xFF00FF00 0xFF0000FF 0xFFFFFFFF 0xFF000000; do
    adb shell am start -a android.intent.action.VIEW -d "data:,$color"
    sleep 2
done

# 4. Brightness Test
echo "[4] Brightness Test:"
adb shell settings put system screen_brightness 255
sleep 2
adb shell settings put system screen_brightness 10
sleep 2
adb shell settings put system screen_brightness 128

echo "[✓] Screen Test Complete!"
