📱 Samsung — Odin

🔷 Samsung Odin

أداة Windows خاصة بسامسونج. الملفات بصيغة .tar أو .tar.md5.

واجهة Samsung Odin — ملاحظة أقسام BL, AP, CP, CSC

الأقسام في Odin:

القسم الملف الوظيفة
BL BOOTLOADER_xxx.tar.md5 Bootloader + Recovery
AP AP_xxx.tar.md5 System + Kernel + Recovery (الملف الأكبر)
CP CP_xxx.tar.md5 Modem / Baseband
CSC CSC_xxx.tar.md5 Carrier Settings + Region
HOME_CSC HOME_CSC_xxx.tar.md5 نفس CSC لكن بدون مسح البيانات

الخطوات:

1
أدخل Download Mode: Volume Down + Home + Power (ثم Volume Up للتأكيد)
⬇
2
افتح Odin → وصّل الهاتف → يجب أن يظهر COM:xx بالأزرق
⬇
3
أضف الملفات: BL → BL, AP → AP, CP → CP, CSC → CSC (أو HOME_CSC)
⬇
4
تأكد من عدم تفعيل Re-partition إلا إذا كان لديك PIT file
⬇
5
اضغط Start → انتظر PASS! بالأخضر
⚠️
ملاحظة: إذا فشل التفليش وظهر FAIL بالأحمر: جرب منفذ USB آخر، أعد تشغيل Odin، أو استخدم إصدار أقدم من Odin.

🔶 Xiaomi — Mi Flash

🔶 Xiaomi Mi Flash Tool

يتطلب Unlock Bootloader أولاً (انتظار 168 ساعة/7 أيام). الملفات بصيغة .tgz.

واجهة Mi Flash Tool — اختر clean all أو save user data

الخطوات:

1
فتح Bootloader عبر Mi Unlock (انتظر 168 ساعة)
⬇
2
حمّل ROM Fastboot من xiaomifirmwareupdater.com
⬇
3
فك ضغط ROM → افتح Mi Flash → اضغط Select واختر مجلد ROM
⬇
4
أدخل Fastboot: Volume Down + Power → اضغط Refresh في Mi Flash
⬇
5
اختر clean all (مسح كامل) أو save user data → اضغط Flash

أوضاع التفليش:

الوضع الاستخدام الطريقة
Fastboot الطريقة العادية Volume Down + Power → Mi Flash
EDL (9008) Hard Brick / Bootloader Locked Test Points + QPST
Recovery تفليش ZIP Volume Up + Power → Install ZIP

🔴 OnePlus

🔴 OnePlus

أسهل الشركات في التفليش. Bootloader سهل الفتح وROMs متاحة بشكل رسمي.

OnePlus — Fastboot Flash 
# 1. فتح Bootloader
$ adb reboot bootloader
$ fastboot flashing unlock

# 2. تحميل Factory Image
# https://www.oneplus.com/support/softwareupgrade

# 3. تفليش (من مجلد Factory Image)
$ fastboot flash boot boot.img
$ fastboot flash dtbo dtbo.img
$ fastboot flash vbmeta vbmeta.img
$ fastboot flash recovery recovery.img
$ fastboot flash system system.img
$ fastboot flash vendor vendor.img

# 4. أو استخدم flash-all (أسهل)
$ ./flash-all.sh   # Linux/macOS
> flash-all.bat    # Windows

# 5. إعادة التشغيل
$ fastboot reboot

🔴 Huawei

🔴 Huawei

أصعب الشركات حالياً. Huawei أغلقت Bootloader رسمياً من 2018. الحلول محدودة.

🚨
تحذير: Huawei لم تعد تقدم أكواد فتح Bootloader. أي أداة تدعي فتح Huawei قد تكون احتيالية!

الحلول المتاحة:

الحالة الحل الصعوبة
Bootloader مفتوح مسبقاً Fastboot flash عادي سهل
Bootloader مغلق + Kirin DC-Phoenix / HCU Client (مدفوع) صعب
Hard Brick Test Points + EDL Mode متقدم
FRP Lock Huawei ID + FRP bypass tools متوسط

🛠️ أدوات تفليش عامة

الأداة الشركات الملاحظات
SP Flash Tool MediaTek (Xiaomi Redmi القديمة، Oppo، Vivo) Scatter file ضروري. خطر Brick إذا اختيار خاطئ!
QPST / QFIL Qualcomm (Samsung، Xiaomi، OnePlus) EDL Mode (9008) + Programmer file
LG Flash Tool LG KDZ files
