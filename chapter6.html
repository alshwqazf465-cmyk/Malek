مشاكل البطارية

تشخيص · معايرة · شحن مباشر · فحص IC الشحن — شرح تفصيلي كامل

🔋 قراءة حالة البطارية عبر ADB

💡
أهم أمر لتشخيص البطارية: adb shell dumpsys battery — هذا الأمر وحده يغنيك عن فتح الهاتف في ٧٠٪ من مشاكل البطارية. يعطيك كل المعلومات التي تحتاجها لتقرر إذا كانت المشكلة في البطارية نفسها أم في البوردة.
الأمر: adb shell dumpsys battery 
adb shell dumpsys battery

# 📋 النتيجة الكاملة مع شرح كل سطر:
Current Battery Service state:
  AC powered: false           # هل الشاحن موصول؟ false = لا
  USB powered: true           # هل كابل USB موصول؟ true = نعم
  Wireless powered: false     # شحن لاسلكي؟ false = لا
  Max charging current: 1500000 # أقصى تيار شحن = 1500mA = 1.5A
  Max charging voltage: 5000000 # أقصى جهد شحن = 5V
  Charge counter: 2800000      # الشحنة المتبقية بالميكرو أمبير
  status: 2                   # حالة الشحن (انظر الجدول أدناه)
  health: 2                   # صحة البطارية (انظر الجدول أدناه)
  present: true               # هل البطارية موجودة؟ true = نعم
  level: 85                   # نسبة الشحن الحالية (0-100)
  scale: 100                  # أقصى نسبة (دائماً 100)
  voltage: 4201               # جهد البطارية بالملي فولت (4201mV = 4.2V)
  temperature: 298            # حرارة البطارية (298 = 29.8°C — اقسم على 10)
  technology: Li-ion          # نوع البطارية

📊 تفسير قيم status (حالة الشحن):

القيمة المعنى متى تظهر؟ الإجراء
1 Unknown حالة غير معروفة (نادر) أعد تشغيل الهاتف وجرب مرة أخرى
2 Charging الهاتف متصل بالشاحن ويشحن ✅ طبيعي — لا شيء مطلوب
3 Discharging الهاتف مفصول من الشاحن ويستهلك ✅ طبيعي — لا شيء مطلوب
4 Not charging متصل بالشاحن لكن لا يشحن ❌ افحص الكابل، الشاحن، IC الشحن
5 Full البطارية ممتلئة ١٠٠٪ ✅ طبيعي — افصل الشاحن

📊 تفسير قيم health (صحة البطارية):

القيمة المعنى الخطورة الإجراء الفوري
1 Unknown ⚠️ منخفضة أعد التشغيل وجرب مرة أخرى
2 Good ✅ لا شيء البطارية سليمة — لا تحتاج شيئاً
3 Overheat 🔥 متوسطة افصل الشاحن فوراً! اترك الهاتف يبرد. افحص IC الشحن
4 Dead 🚨 عالية البطارية تالفة. استبدلها فوراً
5 Over voltage 💥 خطيرة جداً افصل الشاحن فوراً! IC الشحن تالف وقد ينفجر
temperature > 450 حرارة > 45°C 🔥🔥 عالية افصل الشاحن. افحص IC الشحن والبطارية

🔋 تمثيل بصري لحالة البطارية:

85% — جيدة ✅
25% — منخفضة ⚠️
100% — لكن health: 4 (تالفة) 🚨

🔍 أوامر إضافية لفحص البطارية:

أوامر متقدمة لفحص البطارية 
# معرفة السعة الفعلية مقابل السعة الأصلية
adb shell cat /sys/class/power_supply/battery/charge_full
adb shell cat /sys/class/power_supply/battery/charge_full_design
# charge_full = السعة الحالية | charge_full_design = السعة الأصلية
# مثال: 3200000 / 4500000 = 71% (فقدت 29% من عمرها)

# معرفة عدد دورات الشحن
adb shell cat /sys/class/power_supply/battery/cycle_count
# أكثر من 500 دورة = البطارية قاربت نهاية عمرها

# فحص استهلاك البطارية (يتطلب Root)
adb shell su -c "dumpsys batterystats --reset"   # مسح الإحصائيات
adb shell dumpsys batterystats | grep "Estimated power use"
# يعرض أكثر التطبيقات استهلاكاً للبطارية

📊 مشاكل البطارية الشائعة — تشخيص وحلول

المشكلة العرض كيف تتأكد؟ السبب الحل
لا يشحن نهائياً لا LED، لا شعار بطارية، لا استجابة عند توصيل الشاحن جرب شاحن وكابل آخر. إذا لم يستجب: قس البطارية بالملتيميتر. إذا 0V = بطارية تالفة. إذا 3.7V+ = مشكلة بوردة. IC شحن تالف، منفذ USB مكسور، كابل FPC مقطوع، بطارية ميتة ١. نظف منفذ USB
٢. جرب شاحن + كابل آخر
٣. افحص IC الشحن بالملتيميتر
٤. استبدل IC الشحن
يشحن ببطء شديد يستغرق ٥+ ساعات للشحن الكامل استخدم dumpsys battery. انظر إلى Max charging current. إذا كان أقل من 500mA = مشكلة. شاحن رديء، كابل تالف، IC شحن ضعيف، منفذ USB متسخ ١. جرب شاحن 2A+
٢. جرب كابل أصلي
٣. نظف منفذ USB
٤. افحص IC الشحن
يفرغ بسرعة ينتهي في ٢-٣ ساعات بدلاً من يوم كامل افحص cycle_count. إذا > 500 = البطارية منتهية. استخدم BetterBatteryStats للبحث عن Wakelocks. بطارية قديمة، تطبيق Wakelock، IC شحن تالف يسحب تيار ١. افحص استهلاك التطبيقات
٢. احذف التطبيقات المشبوهة
٣. امسح cache
٤. استبدل البطارية
يتوقف عند نسبة معينة يتوقف عند 80% أو 99% ولا يكمل لاحظ هل يتوقف عند نفس النسبة دائماً. إذا نعم = معايرة خاطئة. معايرة خاطئة، IC شحن لا يقرأ الجهد صحيحاً ١. قم بمعايرة البطارية
٢. إذا استمر: افحص IC الشحن
ينتفخ غطاء خلفي مرتفع، الشاشة منفوخة للأعلى المس ظهر الهاتف. إذا كان غير مستوٍ أو تشعر بفراغ = البطارية منتفخة. تفاعل كيميائي داخلي بسبب حرارة أو عيب تصنيع 🚨 استبدل البطارية فوراً! لا تشحنها. لا تضغط عليها. خطر انفجار!
يغلق عند ٢٠-٣٠٪ إطفاء مفاجئ مع أن البطارية لم تصل ٠٪ لاحظ إذا كان يغلق عند نفس النسبة كل مرة. بطارية تالفة فقدت قدرتها، معايرة خاطئة ١. معايرة البطارية
٢. إذا استمر: استبدل البطارية

📊 مخطط تدفق تشخيص مشاكل البطارية

🔋 العميل: "الهاتف لا يشحن"
⬇
السؤال ١: هل الكابل والشاحن يعملان مع هاتف آخر؟
❌ لا → غيّر الكابل/الشاحن! ✅ نعم → انتقل للسؤال ٢
⬇
السؤال ٢: هل منفذ USB نظيف؟
❌ متسخ → نظف بفرشاة + كحول ✅ نظيف → انتقل للسؤال ٣
⬇
السؤال ٣: هل يظهر مؤشر الشحن على الشاشة؟
✅ يظهر → المشكلة ليست في الشحن ❌ لا يظهر → انتقل للسؤال ٤
⬇
السؤال ٤: افتح الهاتف. قس البطارية بالملتيميتر. هل الجهد 3.7V+؟
✅ 3.7-4.2V → البطارية سليمة. المشكلة في البوردة (IC شحن/PMIC) ❌ أقل من 3.3V أو 0V → البطارية تالفة. استبدلها
⬇
✅ تم التشخيص!

🔧 معايرة البطارية — إعادة ضبط عداد الشحن

💡
ما هي معايرة البطارية؟ مع مرور الوقت، قد يخرج "عداد" البطارية في النظام عن الدقة. فمثلاً النظام يظن أن البطارية ٢٠٪ بينما هي في الحقيقة ٥٪. هذا يسبب إطفاء مفاجئ. المعايرة تعيد ضبط هذا العداد ليتطابق مع الواقع.

📋 الطريقة الأولى: المعايرة اليدوية (بدون كمبيوتر)

🔄 دورة الشحن الكاملة (تستغرق ٦-٨ ساعات)

1
استخدم الهاتف حتى ٠٪ ويطفئ نفسه تماماً. لا تشحنه عندما يطلب منك الشحن — استمر في استخدامه حتى يموت.
⬇
2
اشحن حتى ١٠٠٪ بدون استخدام. وصله بالشاحن الأصلي واتركه يشحن. لا تستخدمه أثناء الشحن. اتركه ٢-٣ ساعات حتى يمتلئ تماماً.
⬇
3
اتركه متصلاً ٣٠ دقيقة إضافية بعد أن يصل ١٠٠٪. هذا يضمن أن البطارية مشحونة بالكامل.
⬇
4
استخدمه حتى ٠٪ مرة أخرى بدون شحن. لا تشحنه ولو للحظة. دعه يطفئ نفسه.
⬇
5
اشحن حتى ١٠٠٪ مرة أخيرة. بعد هذه الدورة الكاملة (٠←١٠٠←٠←١٠٠)، تمت المعايرة ✅

💻 الطريقة الثانية: المعايرة عبر ADB (للمحترفين)

معايرة البطارية عبر ADB (يتطلب Root) 
# 📌 هذه الطريقة تمسح إحصائيات البطارية القديمة.
# النظام سيبدأ في جمع إحصائيات جديدة من الصفر.
# يفضل عملها بعد دورة الشحن اليدوية أعلاه.

# ١. اشحن الهاتف إلى ١٠٠٪ (مهم!)

# ٢. مسح إحصائيات البطارية (يتطلب Root)
adb shell su -c "dumpsys batterystats --reset"

# ٣. مسح ملف batterystats (يتطلب Root)
adb shell su -c "rm /data/system/batterystats.bin"
# إذا لم تجد الملف، لا تقلق — بعض الأجهزة لا تستخدمه

# ٤. إعادة التشغيل
adb reboot

# ٥. بعد الإقلاع، استخدم الهاتف عادياً.
# النظام الآن سيتعلم من جديد كيف تتصرف البطارية.

# ⚠️ ملاحظات مهمة:
# - لا تكرر هذه العملية كثيراً. مرة كل ٣-٦ أشهر تكفي.
# - إذا لم تتحسن البطارية بعد المعايرة = البطارية تالفة وتحتاج استبدال.
# - هذه الطريقة لا تصلح بطارية تالفة — فقط تصحح العداد.

⚡ الشحن المباشر — للاختبار فقط!

🚨
تحذير خطير جداً! الشحن المباشر للبطارية بدون IC حماية (الموجود على البوردة) يعرض البطارية لخطر الانفجار والحريق! هذه الطريقة للاختبار السريع فقط في ورشة الصيانة لتحديد إذا كانت المشكلة في البطارية أم في IC الشحن. لا تترك البطارية أبداً بدون مراقبة أثناء الشحن المباشر!

متى تحتاج الشحن المباشر؟

تحتاجه في حالة واحدة: الهاتف لا يشحن نهائياً وتريد أن تعرف إذا كانت المشكلة في البطارية أم في IC الشحن على البوردة.

📋 طريقة الشحن المباشر (للمحترفين فقط):

⚡ خطوات الشحن المباشر (للتشخيص فقط)

1
افصل البطارية عن الهاتف تماماً. اعزل أطرافها. لا تشحنها وهي متصلة بالبوردة!
⬇
2
استخدم باور سبلاي (Power Supply) وليس شاحن عادي. اضبطه على 4.2V و 0.5A كحد أقصى. لا تستخدم فولت أعلى من 4.2V!
⬇
3
وصّل طرف + من الباور سبلاي إلى + البطارية، وطرف - إلى - البطارية. تأكد من القطبية! العكس = انفجار.
⬇
4
راقب التيار. إذا سحبت البطارية تياراً (0.3-0.5A) = البطارية تقبل الشحن ✅. إذا 0A = البطارية ميتة ❌.
⬇
5
اشحنها لبضع دقائق فقط حتى يصل الجهد إلى 3.7V. ثم أعد توصيلها بالهاتف وجرب التشغيل.
الشحن المباشر — المعلومات التقنية 
# معلومات تقنية عن بطاريات Li-ion:
# - الجهد الاسمي: 3.7V
# - جهد الشحن الكامل: 4.2V
# - جهد التفريغ الآمن: 3.0V (لا تنزل أبداً أقل من هذا!)
# - جهد التلف: أقل من 2.5V (البطارية تموت)

# مراحل الشحن الصحيح (الذي يقوم به IC الشحن عادة):
# ١. CC (Constant Current): تيار ثابت حتى يصل الجهد لـ 4.2V
# ٢. CV (Constant Voltage): جهد ثابت 4.2V حتى يقل التيار لـ 0.05C
# ٣. يقطع الشحن تلقائياً

# ⚠️ الشحن المباشر يتجاوز هذه المراحل = خطر!
# ⚠️ لا تشحن أبداً بتيار أكثر من 1A مباشرة
# ⚠️ لا تترك البطارية أبداً بدون مراقبة

🔬 فحص IC الشحن بالملتيميتر

💡
IC الشحن (Charging IC) هو الشريحة المسؤولة عن تنظيم عملية الشحن. إذا تلفت، لن يشحن الهاتف حتى لو كانت البطارية سليمة. فحصها بالملتيميتر هو الخطوة الأولى لتشخيص مشاكل الشحن.
فحص البوردة بالملتيميتر — قياس جهد VBUS و VBAT

📋 نقاط الفحص الأساسية على البوردة:

نقطة الفحص الموقع الجهد الطبيعي إذا كان 0V إذا كان منخفضاً
VBUS دبابيس منفذ USB (الدبوس الأكبر) 5.0V عند توصيل الشاحن كابل/شاحن تالف، منفذ USB مكسور شاحن ضعيف، كابل رديء
VBAT / BATT+ فيصل البطارية (طرف +) 3.7V - 4.2V بطارية ميتة، IC شحن تالف بطارية منخفضة الشحن
VPH_PWR مكثف كبير قرب PMIC 3.7V - 4.2V PMIC تالف، مسار مقطوع مقاومة في المسار
VREG_LDO مكثفات صغيرة قرب PMIC 1.8V أو 3.3V PMIC تالف PMIC تالف جزئياً
VBUCK (CPU) مكثفات قرب المعالج 0.8V - 1.2V PMIC لا يغذي المعالج جهد غير كافٍ = Bootloop

🔍 كيف تفحص IC الشحن خطوة بخطوة:

1
اضبط الملتيميتر على DC 20V. هذا هو وضع قياس الفولت المستمر.
⬇
2
ضع الطرف الأسود على GND (أي نقطة أرضي — الهيكل المعدني أو برغي).
⬇
3
ضع الطرف الأحمر على VBUS (دبوس الشحن في منفذ USB). يجب أن تقرأ 5V مع توصيل الشاحن.
⬇
4
ضع الطرف الأحمر على VBAT (طرف + في فيصل البطارية). يجب أن تقرأ 3.7-4.2V.
⬇
5
إذا كان VBUS = 5V و VBAT = 0V: IC الشحن لا ينقل التيار للبطارية = IC تالف.

🩺 فحص الشورت (القفلة) في IC الشحن:

فحص الشورت في دائرة الشحن 
# 📌 فحص الشورت (Short Circuit):
# ١. اضبط الملتيميتر على وضع "الجرس" (Continuity/Buzzer)
# ٢. ضع طرف على GND والطرف الآخر على VBAT
# ٣. إذا سمعت صفارة مستمرة = يوجد شورت في دائرة الشحن!

# الأماكن التي تفحص فيها الشورت:
# - المكثفات حول IC الشحن (أي مكثف يعطي صفارة = شورت)
# - المكثفات حول PMIC
# - فيصل البطارية (لا يجب أن يكون شورت بين + و -)

# 🔧 إذا وجدت شورت:
# ١. افحص المكثفات واحداً واحداً
# ٢. المكثف التالف = شورت (صفارة على طرفيه)
# ٣. انزعه واختبر مرة أخرى. إذا اختفى الشورت = استبدل المكثف
# ٤. إذا استمر الشورت = IC نفسه تالف ويحتاج استبدال

🔥 تشخيص سخونة البطارية

سخونة البطارية غير طبيعية وتدل على مشكلة خطيرة. هذه جدول لتشخيص مصدر السخونة:

مكان السخونة متى تحدث؟ السبب المحتمل الإجراء
أسفل الهاتف (قرب منفذ الشحن) أثناء الشحن IC شحن تالف أو شاحن بتيار عالي جرب شاحن آخر. إذا استمرت = IC شحن تالف
وسط الهاتف (منطقة المعالج) أثناء الاستخدام الثقيل طبيعي إذا كان الاستخدام ثقيلاً. إذا سخن بدون سبب = شورت في CPU/GPU افحص استهلاك CPU. إذا 100% بدون سبب = تطبيق خبيث
البطارية نفسها طوال الوقت حتى بدون استخدام بطارية تالفة أو IC شحن يسحب تيار زائد افصل الهاتف. إذا استمرت السخونة = بطارية تالفة (استبدلها فوراً)
أعلى الهاتف (منطقة الشبكة) في منطقة إشارة ضعيفة IC شبكة يعمل بأقصى طاقة للبحث عن إشارة طبيعي في المناطق الضعيفة. فعل Airplane Mode للاختبار

🎬 سيناريوهات واقعية

السيناريو ١: بطارية تخلص في ساعتين

🧊 الموقف:

العميل: "البطارية كانت تعمر يوم كامل، فجأة صارت تخلص في ساعتين."

🤔 التشخيص:

  1. افحص صحة البطارية: adb shell dumpsys battery. انظر إلى health. إذا health: 4 = استبدل البطارية.
  2. افحص عدد الدورات: adb shell cat /sys/class/power_supply/battery/cycle_count. إذا > 500 = البطارية انتهى عمرها.
  3. افحص السعة: قارن charge_full مع charge_full_design. إذا فقدت > 30% = استبدلها.
  4. إذا البطارية سليمة: ابحث عن Wakelocks. استخدم adb shell dumpsys batterystats لمعرفة التطبيق المستهلك.

✅ الحل:

في ٧٠٪ = بطارية تحتاج استبدال. في ٢٠٪ = تطبيق Wakelock (احذفه). في ١٠٪ = IC شحن تالف.

السيناريو ٢: الهاتف لا يشحن نهائياً

📡 الموقف:

العميل: "الهاتف ما يشحن أبداً. جربت ٣ شواحن مختلفة."

🤔 التشخيص:

  1. نظف منفذ USB بفرشاة + كحول. هذه الخطوة تحل ٣٠٪ من المشاكل.
  2. جرب الشحن اللاسلكي (إذا كان مدعوماً). إذا شحن = منفذ USB تالف.
  3. افتح الهاتف. قس البطارية بالملتيميتر.
  4. إذا VBAT = 0V: البطارية تالفة. استبدلها.
  5. إذا VBAT = 3.7V: المشكلة في IC الشحن. افحص VBUS (يجب 5V).
  6. إذا VBUS = 5V و VBAT = 3.7V لكن الهاتف لا يشتغل: المشكلة في PMIC أو مسار الطاقة.

✅ الحل:

في ٣٠٪ = تنظيف المنفذ يحل المشكلة. في ٣٠٪ = بطارية تالفة. في ٣٠٪ = IC شحن تالف. في ١٠٪ = PMIC تالف.

سكريبت ١: مراقبة مستمرة للبطارية

battery-monitor.sh — مراقبة وتسجيل حالة البطارية 
#!/bin/bash
# Battery Monitor - يسجل حالة البطارية كل دقيقة
# مفيد لتشخيص مشاكل الاستنزاف السريع أو الشحن البطيء

LOGFILE="battery_log_$(date +%Y%m%d_%H%M%S).csv"
echo "timestamp,level,status,health,voltage,temp,current" > "$LOGFILE"

echo "========================================"
echo "  🔋 Battery Monitor"
echo "  Logging to: $LOGFILE"
echo "  Press Ctrl+C to stop"
echo "========================================"

while true; do
    LEVEL=$(adb shell dumpsys battery | grep "level:" | awk '{print $2}')
    STATUS=$(adb shell dumpsys battery | grep "status:" | head -1 | awk '{print $2}')
    HEALTH=$(adb shell dumpsys battery | grep "health:" | head -1 | awk '{print $2}')
    VOLT=$(adb shell dumpsys battery | grep "voltage:" | awk '{print $2}')
    TEMP=$(adb shell dumpsys battery | grep "temperature:" | awk '{print $2}')
    CURR=$(adb shell dumpsys battery | grep "Max charging current:" | awk '{print $4}')

    echo "$(date '+%H:%M:%S'),$LEVEL,$STATUS,$HEALTH,$VOLT,$TEMP,$CURR" >> "$LOGFILE"

    # عرض مباشر
    echo "$(date '+%H:%M:%S') | Level: ${LEVEL}% | Volt: ${VOLT}mV | Temp: $((TEMP/10))°C"

    # تنبيه إذا الحرارة > 45°C
    if [ "$TEMP" -gt 450 ]; then
        echo "[!] 🚨 WARNING: Battery temperature > 45°C! ($((TEMP/10))°C)"
        echo "[!] Disconnect charger immediately if charging!"
        # يمكنك إضافة أمر لإطفاء الشاشة تلقائياً:
        # adb shell input keyevent 26
    fi

    # تنبيه إذا health = 3 (حرارة زائدة)
    if [ "$HEALTH" = "3" ]; then
        echo "[!] 🔥 WARNING: Battery health = OVERHEAT!"
    fi

    # تنبيه إذا health = 4 (تالفة)
    if [ "$HEALTH" = "4" ]; then
        echo "[!] 💀 CRITICAL: Battery health = DEAD! Replace immediately!"
    fi

    sleep 60
done

سكريبت ٢: تشخيص سريع للبطارية

battery-check.sh — فحص سريع للبطارية 
#!/bin/bash
# Battery Quick Check - تقرير سريع عن حالة البطارية

echo "========================================"
echo "  🔋 Battery Quick Check"
echo "========================================"
echo ""

# معلومات أساسية
LEVEL=$(adb shell dumpsys battery | grep "level:" | awk '{print $2}')
HEALTH=$(adb shell dumpsys battery | grep "health:" | head -1 | awk '{print $2}')
TEMP=$(adb shell dumpsys battery | grep "temperature:" | awk '{print $2}')
VOLT=$(adb shell dumpsys battery | grep "voltage:" | awk '{print $2}')

echo "📊 Basic Info:"
echo "   Level: ${LEVEL}%"
echo "   Voltage: ${VOLT}mV ($(echo "scale=1; $VOLT/1000" | bc)V)"
echo "   Temperature: $((TEMP/10))°C"

# تفسير health
echo ""
echo "🩺 Health Check:"
case $HEALTH in
    2) echo "   ✅ Battery is GOOD" ;;
    3) echo "   🔥 Battery is OVERHEATING!" ;;
    4) echo "   💀 Battery is DEAD - Replace now!" ;;
    5) echo "   ⚡ OVER VOLTAGE - Charging IC damaged!" ;;
    *) echo "   ❓ Unknown ($HEALTH)" ;;
esac

# السعة والاستهلاك
echo ""
echo "📈 Capacity Check:"
FULL=$(adb shell cat /sys/class/power_supply/battery/charge_full 2>/dev/null || echo "N/A")
DESIGN=$(adb shell cat /sys/class/power_supply/battery/charge_full_design 2>/dev/null || echo "N/A")
if [ "$FULL" != "N/A" ] && [ "$DESIGN" != "N/A" ]; then
    PERCENT=$(echo "scale=1; $FULL * 100 / $DESIGN" | bc)
    echo "   Full: ${FULL}μAh"
    echo "   Design: ${DESIGN}μAh"
    echo "   Health: ${PERCENT}%"
    if [ "$(echo "$PERCENT < 70" | bc)" = "1" ]; then
        echo "   ⚠️ Battery capacity below 70% - consider replacement"
    fi
else
    echo "   Capacity info not available (needs root on some devices)"
fi

# دورات الشحن
echo ""
echo "🔄 Cycle Count:"
CYCLES=$(adb shell cat /sys/class/power_supply/battery/cycle_count 2>/dev/null || echo "N/A")
echo "   Cycles: $CYCLES"
if [ "$CYCLES" != "N/A" ] && [ "$CYCLES" -gt 500 ]; then
    echo "   ⚠️ Over 500 cycles - battery approaching end of life"
fi

echo ""
echo "[✓] Check complete!"

💎 نصائح ذهبية للبطارية

✅
قواعد لا تنساها:
  1. لا تشحن الهاتف طوال الليل. الشحن الزائد يولد حرارة تقصر عمر البطارية. IC الشحن يقطع الشحن عند ١٠٠٪ لكن الحرارة تستمر.
  2. لا تترك البطارية تصل إلى ٠٪. التفريغ العميق (أقل من 3.0V) يدمر خلايا البطارية بشكل دائم.
  3. استخدم الشاحن والكابل الأصليين. الشواحن الرخيصة ترسل تياراً غير مستقراً يدمر IC الشحن مع الوقت.
  4. إذا انتفخت البطارية، استبدلها فوراً. لا تشحنها. لا تضغط عليها. لا تثقبها. البطارية المنتفخة قنبلة موقوتة.
  5. لا تشحن الهاتف تحت الوسادة. الحرارة هي العدو الأول للبطارية. اشحن الهاتف في مكان مفتوح.
  6. افحص IC الشحن إذا تكرر تلف البطاريات في نفس الهاتف. IC تالف = يقتل البطارية الجديدة أيضاً.
  7. البطاريات البديلة الرخيصة خطيرة. استخدم بطاريات أصلية أو من ماركات معروفة.
