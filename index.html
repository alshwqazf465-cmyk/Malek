٢ تشريح نظام أندرويد

🧱 هرم أندرويد (من الأسفل للأعلى)

┌─────────────────────────────┐
│       التطبيقات (Apps)       │  ← فيسبوك، واتساب...
├─────────────────────────────┤
│     إطار العمل (Framework)   │  ← يدير النوافذ والإشعارات
├─────────────────────────────┤
│       المكتبات (Libraries)   │  ← SQLite, WebKit, OpenGL
├─────────────────────────────┤
│    بيئة التشغيل (Runtime)    │  ← ART (Android Runtime)
├─────────────────────────────┤
│       النواة (Kernel)        │  ← Linux Kernel
├─────────────────────────────┤
│      الهاردوير (Hardware)    │  ← المعالج، الذاكرة، الشاشة
└─────────────────────────────┘

🧩 أقسام الذاكرة (Partitions)

القسمالوظيفة
/bootملفات الإقلاع والنواة (Kernel)
/systemنظام أندرويد الأساسي
/recoveryوضع الاسترداد
/dataبيانات المستخدم والتطبيقات
/cacheملفات مؤقتة
/modemبرنامج المودم والراديو
/efsمعلومات حساسة (IMEI، شهادات)
/persistإعدادات دائمة (WiFi، BT)
/vendorتعريفات خاصة بالجهاز

شكل (١): طبقات وهيكل نظام أندرويد

🔄 دورة الإقلاع (Boot Process)

  1. زر التشغيل ← كهرباء من البطارية/PMIC
  2. Boot ROM ← فحص Bootloader
  3. Bootloader ← تحميل النواة
  4. Kernel ← تشغيل التعريفات
  5. Init ← خدمات النظام
  6. Zygote ← تهيئة ART
  7. System Server ← النوافذ والإشعارات
  8. Launcher ← ظهور الشاشة الرئيسية ✅
⚠️ تشخيص: توقف عند ٣ = مشكلة Bootloader، توقف عند ٤-٦ = Bootloop غالباً.

٣ أداة ADB

🔌 ما هو ADB؟

جسر أوامر بين الكمبيوتر والهاتف، يسمح بإرسال أوامر والهاتف شغال أو في Recovery.

🛠️ التحضير (كل الماركات)

ويندوز - تثبيت Platform Tools 
1. حمل Platform Tools من موقع أندرويد الرسمي
2. فك الضغط في C:\adb
3. افتح CMD واكتب:
cd C:\adb

📱 تفعيل خيارات المطور:

  1. الإعدادات > حول الهاتف
  2. اضغط رقم الإصدار ٧ مرات
  3. الإعدادات > خيارات المطور
  4. فعل تصحيح أخطاء USB

📋 أوامر ADB الأساسية

أوامر ADB 
adb devices              # عرض الأجهزة المتصلة
adb reboot               # إعادة تشغيل
adb reboot recovery      # دخول الريكفري
adb reboot bootloader    # دخول الفاست بوت
adb shell                # فتح نافذة أوامر داخل الهاتف
adb install app.apk      # تثبيت تطبيق
adb pull /path/file      # سحب ملف من الهاتف
adb push file /path/     # إرسال ملف للهاتف
adb logcat               # سجل النظام للتشخيص
adb bugreport            # تقرير تشخيصي كامل
أوامر متقدمة مفيدة 
# معرفة حالة البطارية
adb shell dumpsys battery

# معرفة نسخة الأندرويد
adb shell getprop ro.build.version.release

# قائمة بكل خصائص الجهاز
adb shell getprop

# إعادة تشغيل الواجهة (لحل تجمد الشاشة)
adb shell am start -a android.intent.action.MAIN \
  -c android.intent.category.HOME

# مسح كاش تطبيق معين
adb shell pm clear com.example.app

٤ أداة Fastboot

⚡ ما هو Fastboot؟

بروتوكول يشتغل قبل نظام أندرويد، يستخدم لتفليش ملفات النظام وتعديل الأقسام. يعمل فقط في وضع Bootloader.

📥 الدخول إلى Fastboot حسب الماركة

الماركةطريقة الدخول (والهاتف مطفأ)
Samsung زر خفض الصوت + تشغيل (Download Mode، يستخدم Odin)
Xiaomi زر خفض الصوت + تشغيل
OnePlus زر رفع الصوت + تشغيل
Huawei زر خفض الصوت + تشغيل + توصيل USB
⚠️ سامسونج خاصة: لا تستخدم Fastboot! تستخدم Download Mode مع برنامج Odin.

📋 أوامر Fastboot الأساسية

أوامر Fastboot 
fastboot devices              # التحقق من اتصال الهاتف
fastboot reboot               # إعادة تشغيل
fastboot oem unlock           # فتح البوت لودر (يمسح البيانات!)
fastboot oem lock             # إغلاق البوت لودر
fastboot flash [قسم] [ملف]   # تفليش ملف إلى قسم
fastboot boot [ملف.img]       # تشغيل ريكفري مؤقت
fastboot getvar all           # عرض معلومات الجهاز
مثال: تفليش ريكفري لـ Xiaomi 
fastboot flash recovery twrp.img
fastboot reboot recovery

٥ تشخيص مشاكل الشبكة والـ IMEI

📶 مسار الاتصال في الهاتف

شريحة SIM → قارئ الشريحة → مودم (Baseband) ← EFS/NVRAM/QCN
                                              ↓
                                         نظام أندرويد ← عرض الأبراج

🔍 سلم التشخيص الذهبي للشبكة

1 هل الشريحة مقروءة؟ لا ← نظف الشريحة أو جرب أخرى
2 هل IMEI موجود؟ لا ← *#06# يظهر null/0 ← EFS تالف
3 هل Baseband موجود؟ لا ← إعدادات > حول الهاتف = "غير متوفر"
4 هل يبحث عن شبكات؟ لا ← مشكلة مودم أو أنتين
5 هل يسجل على الشبكة؟ لا ← مشكلة EFS/شهادة
6 الهاردوير آي سي الشبكة أو باور

📱 أكواد فحص الشبكة حسب الماركة

Samsung 
*#0011#    # معلومات الشبكة الحية
*#06#      # عرض IMEI
*#0808#    # إعدادات USB
Xiaomi 
*#*#6484#*#*   # قائمة اختبار المصنع (CIT)
*#*#4636#*#*   # معلومات الهاتف والشبكة
OnePlus / Huawei 
*#06#             # IMEI (OnePlus)
*#*#2846579#*#*   # Project Menu (Huawei)
عبر ADB (لأي جهاز) 
# فحص Baseband
adb shell getprop gsm.version.baseband

# عرض IMEI
adb shell service call iphonesubinfo 1

٦ مشاكل التشغيل والبطارية (سوفتوير)

🪫 الهاتف لا يشحن / شحن وهمي

  1. نظف البورت (فرشاة ناعمة + كحول أيزوبروبيل)
  2. جرب سلك وشاحن معروفين
  3. اتركه بالشاحن 30 دقيقة (بطارية مستنزفة)
  4. ادخل Recovery وامسح الكاش
  5. معايرة البطارية (بحاجة Root): 
    adb shell
su
rm /data/system/batterystats.bin
reboot

🔁 Bootloop (شعار متكرر)

الأسباب: تحديث فاشل، تطبيق خبيث، ملف System تالف.

  1. ادخل Recovery (رفع صوت + تشغيل)
  2. Wipe Cache Partition
  3. إذا لم ينفع: Wipe Data/Factory Reset (يمسح كل شيء!)
  4. إذا استمر: فلاش روم رسمي

٧ الرومات وأنواعها والتفليش

📦 أنواع الرومات

النوعالوصفالاستخدام
Stock ROMروم المصنع الرسميإصلاح مشاكل النظام
Custom ROMروم معدل (LineageOS)تحديثات، مميزات إضافية
Fastboot ROMبصيغة imgXiaomi/OnePlus
Recovery ROMبصيغة zipتحديثات OTA
Odin ROMبصيغة tar.md5خاص بسامسونج

🔧 التفليش حسب الماركة

1. Samsung – عبر Odin

  1. حمل Odin وروم رسمي (من SamMobile)
  2. ادخل Download Mode (خفض صوت + تشغيل)
  3. الروم يحتوي: BL, AP, CP, CSC
  4. ضع كل ملف في مكانه واضغط Start

2. Xiaomi – عبر Mi Flash Tool

  1. حمل Mi Flash Tool وروم Fastboot
  2. ادخل Fastboot
  3. اختر مسار الروم واختر Clean All
  4. اضغط Flash

3. OnePlus – Fastboot / MSM Tool

MSM Tool خاصة بكوالكوم، تعيد الهاتف من Hard Brick.

4. Huawei – Hisuite / DC-Phoenix

Hisuite رسمي، DC-Phoenix مدفوع للإصلاح العميق.

٨ أساسيات الهاردوير للمبتدئين

⚡ مفاهيم كهربائية

المفهومالرمزالوحدةتشبيه
الجهدVفولتضغط الماء
التيارIأمبيركمية الماء
المقاومةRأوم Ωضيق الأنبوب

🧱 المكونات الأساسية

  • مكثف: يخزن شحنة - إذا انتفخ = شورت
  • مقاومة: تقلل التيار - إذا كانت مفتوحة = تالفة
  • ملف: ينظم التيار - إذا OL = تالف
  • PMIC: يدير توزيع الطاقة
  • Modem IC: مسؤول الشبكة
  • Charging IC: تنظيم الشحن

🩺 فحص الشورت (القفلة)

  1. ضع الملتيميتر على وضع الجرس
  2. طرف على GND وطرف على نقطة القياس
  3. سمعت صفارة = القطعة شورت

٩ أدوات الصيانة المادية

🧰 عدة المبتدئين

  1. مفكات دقيقة (Phillips #00, #000)
  2. ملتيميتر رقمي (DT830D)
  3. باور سبلاي (0-15V/2A)
  4. كاوية لحام 60W
  5. سلك لحام + فلوكس
  6. عدسة مكبرة برأس
  7. فرشاة ناعمة + كحول أيزوبروبيل 99%
  8. ملقط (Tweezers)
  9. سجادة مضادة للكهرباء الساكنة
  10. أكواب لحفظ البراغي
