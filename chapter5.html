مشاكل الشبكة وIMEI

استعادة IMEI · Baseband · فتح قفل الشبكة · حلول Qualcomm وMediaTek

📡 تشخيص مشاكل الشبكة

فحص الشبكة عبر ADB 
# 1. التحقق من IMEI
$ adb shell service call iphonesubinfo 1
Result: Parcel( 0x00000000: '000000000000000' )  <-- IMEI مفقود!

# 2. التحقق من Baseband
$ adb shell getprop gsm.version.baseband
Unknown  <-- Baseband مفقود!

# 3. حالة الشبكة
$ adb shell dumpsys telephony.registry | grep -E "mServiceState|mSignalStrength"

# 4. معلومات SIM
$ adb shell dumpsys iphonesubinfo

# 5. APN Settings
$ adb shell content query --uri content://telephony/carriers --projection name,apn

🔧 استعادة IMEI

⚠️
تنبيه قانوني: تغيير IMEI غير قانوني في معظم الدول. هذا الدليل يشرح استعادة IMEI الأصلي بعد فقدانه فقط!

السبب 1: تلف EFS بعد تفليش خاطئ

الحل: استعادة نسخة احتياطية من EFS (إن وجدت)

استعادة EFS 
# 1. إذا كان لديك نسخة احتياطية
$ adb push efs_backup.img /sdcard/
$ adb shell su -c "dd if=/sdcard/efs_backup.img of=/dev/block/bootdevice/by-name/efs"
$ adb reboot

# 2. إذا لم يكن لديك نسخة — استخدم NV Backup (Qualcomm)
$ adb shell su -c "cat /persist/radio/qcril_database/qcril.db" > qcril_backup.db

السبب 2: فقدان IMEI على Qualcomm

استخدام QPST/QFIL لكتابة NV Items:

QPST — كتابة IMEI عبر NV Items 
# 1. ادخل EDL Mode (9008)
#    - Qualcomm: Test Points أو adb shell "reboot edl"

# 2. افتح QPST Software Download
#    - Select Programmer: prog_emmc_firehose_xxxx.mbn
#    - Load XML: rawprogram0.xml
#    - Patch: patch0.xml

# 3. أو عبر QFIL (أسهل)
#    - Select Flat Build
#    - Browse → Programmer → prog_emmc_firehose.mbn
#    - Load XML → rawprogram0.xml + patch0.xml
#    - Download

# 4. IMEI يُكتب عبر NV Item 550 (IMEI1) و 551 (IMEI2)
#    استخدم QPST NV Tool:
#    - QPST → Start Clients → NV Tool
#    - NV Item 550: أدخل IMEI (hex format)

السبب 3: فقدان IMEI على MediaTek

MediaTek — IMEI Restore 
# 1. عبر Engineering Mode (إذا كان متاحاً)
$ adb shell am start -n com.mediatek.engineermode/.EngineerMode
#    → CDS Information → Radio Information

# 2. عبر AT Commands (يتطلب Root + Terminal Emulator)
$ su
# echo 'AT+EGMR=1,7,"YOUR_IMEI_1"' > /dev/radio/atci0
# echo 'AT+EGMR=1,10,"YOUR_IMEI_2"' > /dev/radio/atci0

# 3. عبر Maui META (أداة MediaTek الرسمية)
#    - يتطلب Preloader Mode
#    - IMEI Download → Enter IMEI → Write

🔥 سكريبت إصلاح شامل

fix-network.sh — إصلاح مشاكل الشبكة 
#!/bin/bash
# Network Fix Script - Requires Root

echo "[*] Network Fix Script"
echo "[*] Checking device status..."

# Check IMEI
IMEI=$(adb shell service call iphonesubinfo 1 | grep -o '[0-9]*' | tr -d '\n')
echo "[*] IMEI: $IMEI"

if [ "$IMEI" = "000000000000000" ]; then
    echo "[!] IMEI is NULL! EFS may be corrupted."
    echo "[!] Try restoring EFS backup or flashing stock ROM."
    exit 1
fi

# Reset network settings
echo "[*] Resetting network settings..."
adb shell settings put global airplane_mode_on 1
adb shell am broadcast -a android.intent.action.AIRPLANE_MODE
sleep 2
adb shell settings put global airplane_mode_on 0
adb shell am broadcast -a android.intent.action.AIRPLANE_MODE

# Reset APN
echo "[*] Resetting APN..."
adb shell content delete --uri content://telephony/carriers/restore

echo "[✓] Done! Reboot device now."
adb reboot
