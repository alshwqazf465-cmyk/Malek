## 🚀 الرفع على GitHub Pages

1. أنشئ repository جديد على GitHub
2. ارفع كل الملفات (index.html + chapter1-10.html + images/)
3. اذهب إلى Settings → Pages
4. اختر Branch: main → Folder: / (root)
5. انتظر 2 دقائق → ستحصل على رابط موقعك!
