Hard Brick · FRP · Bootloader Locked — الحلول النهائية

💀 Hard Brick (جهاز ميت كلياً)

🚨
Hard Brick = لا شعار، لا شحن، لا Fastboot، لا شيء. الجهاز يبدو "ميتاً" تماماً.

الأسباب:

  • تفليش Bootloader خاطئ
  • قطع الكهرباء أثناء تفليش Partition حرج
  • تلف eMMC/UFS (ذاكرة التخزين الداخلية)
  • تلف PMIC (لا يُخرج جهد)

الحلول حسب المعالج:

المعالجالوضعالأداةالخطوات
Qualcomm EDL Mode (9008) QPST / QFIL Test Points → USB 9008 → Load Programmer → Flash
MediaTek Preloader Mode SP Flash Tool Volume Down + Power → Scatter File → Download
Samsung Exynos Download Mode Odin Volume Down + Bixby + Power → Flash Full ROM
Huawei Kirin USB COM 1.0 DC-Phoenix / HCU Test Points + Tool مدفوع

EDL Mode (Qualcomm 9008):

EDL Mode Entry Methods 
# الطريقة 1: عبر ADB (إذا كان الجهاز يعمل جزئياً)
$ adb shell su -c "reboot edl"

# الطريقة 2: عبر Fastboot
$ fastboot oem edl

# الطريقة 3: Test Points (الأكثر شيوعاً)
# 1. افتح الجهاز
# 2. اعثر على Test Points المؤشرة (عادةً بجانب eMMC)
# 3. وصّل نقطتين معاً بملقط معدني
# 4. وصّل كابل USB
# 5. يجب أن يظهر "Qualcomm HS-USB QDLoader 9008" في Device Manager

# الطريقة 4: Deep Flash Cable (للأجهزة المستعصية)
# كابل مخصص يُجبر الجهاز على الدخول لـ EDL

🔐 FRP (Factory Reset Protection)

شاشة FRP Lock — تطلب حساب Google السابق
⚠️
تنبيه: FRP حماية قانونية ضد السرقة. هذا الدليل يهدف لمساعدة المالك الشرعي فقط!

الحلول القانونية:

الطريقةالوصفالصعوبة
حساب Google الأصلي أدخل البريد وكلمة المرور السابقين سهل (إذا تذكرتها)
Samsung Find My Mobile إذا كان الجهاز مرتبط بحساب Samsung سهل
Google Account Recovery استعادة الحساب عبر google.com/accounts/recovery متوسط
Receipt + Service Center أثبت الملكية لمركز خدمة معتمد سهل (لكن يتطلب وقت)

الحلول التقنية (للأجهزة القديمة):

FRP Bypass Methods (Old Android Versions) 
# Android 5-6: TalkBack Method
# 1. اضغط مطولاً على الشاشة لتفعيل TalkBack
# 2. ارسم حرف L (إلى أسفل ثم يمين)
# 3. افتح Global Context Menu
# 4. افتح TalkBack Settings → Help & Feedback
# 5. افتح YouTube → افتح Chrome → حمل APK

# Android 7-8: QuickShortcutMaker
# 1. ادخل منفذ SIM بطاقة NFC
# 2. افتح Google Assistant → Search
# 3. افتح Chrome → حمل QuickShortcutMaker
# 4. ابحث عن "Google Account Manager"
# 5. Type Email and Password → أضف حساباً جديداً

# Android 9+: أصعب بكثير، يتطلب:
# - Samsung: Combination Firmware
# - Qualcomm: EDL Mode + FRP reset
# - MediaTek: SP Flash Tool + Format FRP partition
